Third Party uses of Ptplot

To have your link added to this list, send email to ptplot at ptolemy eecs berkeley edu

UC Berkeley EECS 20N, Structure & interpretation of Signals & Systems
http://ptolemy.eecs.berkeley.edu/eecs20/berkeley/
Ptolemy II - a Java-based component assembly framework with a graphical user interface called Vergil. These applets tend to be large, so they work best under a fast connection
http://ptolemy.eecs.berkeley.edu/ptolemyII/ptIIlatest
Sphlow
Applet used to generate data for metallurgy.
http://www.ctcms.nist.gov/~powell/Java/
Transmission Line Calculator
http://fermi.la.asu.edu/w9cf/tran/
Differential Evolution (DE) for Continuous Function Optimization
http://www.icsi.berkeley.edu/~storn/code.html
JMM - Java Multi Meter
http://www.agri-vision.nl/CMS/index.php?option=com_content&view=article&id=13&Itemid=80
Digital Filter Design Software "Fiwiz"
http://www.icsi.berkeley.edu/~storn/fiwiz.html#man
Twin Paradox
http://alan.datdec.com/twinparadox/index.html
Debrief - an Open Source application for analysis of Naval Exercises - Uses Ptplot for plotting relationships between vessels
http://www.debrief.info
CollabCAD is a Java based distributed 2D&3D CAD/CAM and CAE software system
http://www.collabcad.gov.in
Filter Design and Visualization - It is in beta and designs n-order butterworth and chebyshev filters as well as resonators with user specified Q. The user may specify low pass, band pass or high pass realization. This applet may not work under Netscape 4.x, try Netscape 6.x instead.
http://www.metrak.com/filter/PlotApplet.html
Scripps Orbit and Permanent Array Center (SOPAC) uses Ptplot to display position time series of continuous GPS site data.
SOPAC sample link:
http://sopac.ucsd.edu/cgi-bin/refinedSeriesAppletWrapper.cgi?site=fvpk&type=flt&rmTJ=yes

Other agencies that process and display GPS site time series also use the ptplot base classes:

USGS-Pasadena
http://pasadena.wr.usgs.gov/scign/Analysis/index.html (click on Java Time Series)
SIMPROCESS
SIMPROCESS "is a hierarchical and integrated process simulation tool that combines Process Mapping, Discrete-Event Simulation, and Activity Based Costing (ABC) in a single easy to use tool.". SIMPROCESS uses Ptplot, see http://www.simprocess.net/products/customizable_real_time_plots.html
Zephyr
Zephyr "is a script-driven data acquisition and measurement automation program written in Java. ... [Zephyr] may be suitable for many other applications involving measurement instruments and actuators, especially where there's a need to change the setup often."
CircuitEngine
CircuitEngine - Circuit simulation applet and application. As of 1/10, the applet demo required Java 6.0. To see Ptplot, create a RC circuit that includ\ es an ammeter and then right click on the meter and select graph.
MAPP Online Pro
Meyer Sound's MAPP Online Pro vvu uses Ptplot. "MAPP Online Pro is a powerful, cross-platform, Java-based application for accurately predicting the coverage pattern, frequency response, impulse response, and maximum SPL output of single or arrayed Meyer Sound loudspeakers."
Magnetometer Data Display
The Magnetometer Data Display from IPS Radio and Space Services, Bureau of Meteorology, Australia, uses PtPlot to display data sets.
