Derivative

Demonstration of the Derivative actor. This model shows that the Derivative actor is able to handle discontinuous inputs, and that when put in cascade with an Integrator actor, the combination is an identity function.

Derivative is available as:

  • As an applet.
The Java Web Start version is preferred, though browsers with versions of Java earlier than Java 1.6.0_13 may only work with the applet version. See Web Start Help for details.
