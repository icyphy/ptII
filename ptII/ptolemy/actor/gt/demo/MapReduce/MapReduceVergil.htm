MapReduce

This page contains an applet that should appear as a separate window.

This model uses transformation attributes to demonstrate the design workflow of a configurable model. The original model (with 1 Map and 1 Reduce) serves as a minimal template that can be used to generate models of arbitrary size. The ConfigureModel attribute configures the template to generate such a model. Its parameters can be edited with "Customize" -> "Configure" from the popup menu. Its mapCount parameter controls the number of total Map actors in the configured model, and its reduceCount controls the number of Reduces. CleanUpModel is another transformation attribute to clean up a configured model to restore the template in the editor. CheckConsistency checks the structural consistency.
