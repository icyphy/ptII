PetriNetSimple

As the model is running, the PetriNet Director sends messages to the PetriNetDisplay. The messages contain the token count for each Place at each iteration.

PetriNetSimple is available as:

  • As an applet.
The Java Web Start version is preferred, though browsers with versions of Java earlier than Java 1.6.0_13 may only work with the applet version. See Web Start Help for details.
