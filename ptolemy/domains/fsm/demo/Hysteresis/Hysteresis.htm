Hysteresis demo with Hierarchical FSM's and SDF

The following plot shows the noisy input signal (blue) and the output signal (red). The noisy input signal consists of a sinusoid plus a white Gaussian noise source.

One possible application of hysteresis is a heater control, where the noisy input signal (blue) could be the input from a temperature sensor. The output signal (red) would then correspond to the heater on/off, with +1 denoting the on state and -1 denoting the off state.
