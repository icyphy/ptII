Help for the Vergil Graph Editor

  • Drag objects from the palette on the left to the graph on the right.
  • External ports and relations can be created from the toolbar.
  • Control-click to create new relations.
  • Drag connections from one port to another. Control-drag to create a connection starting at a relation or an external port.
  • Single ports are solid, Multiports are hollow.
  • To broadcast a value from a single port, connect to a diamond relation.
  • Right click for context menus.
  • Configure external ports from Background Context Menu -> configure ports.
  • Hierarchical models are outlined in red. To create a hierarchical model, drag a composite from the utilities and look inside.
  • To execute, view the Run Window.
