License for ASM Java bytecode manipulation and analysis framework

ASM is used by the standalone version of Nashorn.

Nashorn is the JavaScript engine used by accessors. Oracle JDK 8 through 14 include a builtin version of Nashorn. Oracle JDK 15 and later need to use a standalone Nashorn.

asm-all-5.2.jar was downloaded from https://mvnrepository.com/artifact/org.ow2.asm/asm-all/5.2 on August 27, 2021.

To update asm-all-*.jar:

  1. Download the most recent asm-all-*.jar https://mvnrepository.com/artifact/org.ow2.asm/asm-all
  2. Edit $PTII/configure.in and run: 
          cd $PTII
      autoconf
      ./configure
      ant clean
      ant
  3. Edit $PTII/.classpath.default and $PTII/mk/jnlp.mk and update the version number of asm
  4. Check in your changes. 
          cd $PTII
      git rm -f lib/asm-all-old.old.jar
      git commit -m "Updated asm to asmp-x.y.z.jar" configure.in lib/asm-all-.old.old.jar lib/asm-all-x.y.jar lib/asm-license.htm .classpath.default
      autoconf
      git  commit -m "configure.in changed." configure
