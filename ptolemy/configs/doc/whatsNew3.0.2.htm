Quick Tour of What's New in Version 3.0.2

  • Slider Parameter and Port Parameters.
  • Calculating Fourier series coefficients of a sketched signal.
  • Spectrum of audio picked up from the microphone.
  • Component interaction (CI) domain.
  • Improved discontinuities in continuous-time models.
  • Improved event detection in continuous-time models.
  • Improved sampling of discontinuous signals in continuous-time models.
  • Functions and higher-order functions in the expression language.
  • Early version of higher-order components.
  • Use of higher-order components to realize Goertzel DFT algorithm.
  • Dynamic control of icon location, visible parameters, and MonitorValue.
  • Improved modal models
  • Illustration of timing paradox in asynchronous periodic tasks.
  • Transient states in finite-state machines.
  • Transition refinements in finite-state machines.
  • Illustration of state tracking in finite-state machines.
  • Graphical sticky masses hybrid system. Requires Java 3D, see the GR domain.
  • Trivial use of PythonScript actor.
  • Use of Python and an interactive text shell (Does not work under Web Start).
