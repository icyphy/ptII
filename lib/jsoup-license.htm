License for jsoup HTML parser

jsoup is used by the export HTML facility in $PTII/ptolemy/vergil/basic/export/html.

jsoup-1.14.2 was downloaded from http://jsoup.org on August 24, 2021.

To update jsoup:

  1. Download the most recent jsoup-x.y.z.jar from http://jsoup.org
  2. Edit $PTII/configure.in and run: 
          cd $PTII
      autoconf
      ./configure
      ant clean
      ant
  3. Edit $PTII/.classpath.default and $PTII/mk/jnlp.mk and update the version number of jsoup
  4. Check in your changes. 
          cd $PTII
      git rm -f lib/jsoup-old.old.old.jar
      git commit -m "Updated jsoup to jsoup-x.y.z.jar" configure.in lib/jsoup-old.old.old.jar lib/jsoup-x.y.z.jar lib/jsoup-license.htm .classpath.default
      autoconf
      git  commit -m "configure.in changed." configure
