Plotting Binary Data

This page includes an applet, which may or may not run for you. See Troubleshooting for details.

The plot below obtains its data from two binary files given by a URL address. This binary format is used in pxgraph, the X11 plotting tool that is part of the Ptolemy Project. For information about the binary format, see the PxgraphParser Java class documentation. For information about endian issues, see the Endian Examples.
