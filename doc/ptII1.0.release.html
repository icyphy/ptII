Ptolemy II version 1.0 Release Notes

Ptolemy II version 1.0 is a release of the Ptolemy II Java classes and the Ptolemy II design document

This release is meant for programmers already familiar with Java.

Ptolemy II 1.0 includes applet demonstrations of the domains, but it does not include a GUI.

The design document describes the Ptolemy II design and the implementation of the Java classes.

The Ptolemy design document is shipped with Ptolemy 1.0, it can be found in $PTII/doc/design. The most recent copy of the Ptolemy II design document can be found online at http://ptolemy.eecs.berkeley.edu/ptolemyII/doc/design.html

Platforms

The core of Ptolemy II 1.0 is 100% Java, so it should work on any platform that has JDK1.1. (Note that the GUI shipped with the next release (Ptolemy II 1.0) will use Tcl/Tk and Itcl, so it will not be 100% Java).

We developed Ptolemy II 1.0 under Solaris 2.5.1 and NT4.0sp3 with JDK1.1.6.

Ptolemy II can be built under JDK1.2, though we have not thoroughly tested Ptolemy II under JDK1.2.

See the Tcl Blend section of the installation page for information about the Ptolemy II test suite

Limitations

  • Ptolemy II version 1.0 does not include a GUI.
  • The ptolemy.kernel.event package is likely to change substantially.
