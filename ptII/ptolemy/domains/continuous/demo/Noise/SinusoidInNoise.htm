SinusoidInNoise

This model measures the power spectrum of bandlimited noise added to a sinusoid. The spectrum is plotted together with the theoretical value of the spectrum of the noise. Notice that the sidebands decay slightly more slowly than the theory predicts because of aliasing distortion. You can experiment with the bandwidth of the noise and the frequency and amplitude of the sine wave.

SinusoidInNoise is available in two formats:

  • As a Web Start Application:
  • As an applet.
The Java Web Start version is preferred, though browsers with versions of Java earlier than Java 1.6.0_13 may only work with the applet version. See Web Start Help for details.
