Demo of DE simulation applet

DE Simulation:

Note that this requires that private members of CrossRefList be made public, or else everything has to be compiled under jdk 1.2.

If you were able to run applets, you would have a waveform demo here.
This applet is implemented by ClockApplet.java.

The simulation consists of a Clock Actor, a Poisson Actor, and a WaitingTime Actor connected to a Plot Actor.
