System Design, Modeling and Simulation using Ptolemy II

The most comprehensive documentation about Ptolemy II can be found in the book:

Claudius Ptolemaeus, Editor. "System Design, Modeling, and Simulation using Ptolemy II," Ptolemy.org, 2014.

The book is available as a free PDF download and as a low-cost paperback.

The PDF is included in the release as $PTII/doc/books/systems/PtolemyII_DigitalV1_02.pdf, but the PDF is not present in the SVN developer tree.

The book website includes the book as an entire pdf, as per-chapter pdfs and includes on-line demos.

Please cite the book as:
Claudius Ptolemaeus, Editor, System Design, Modeling, and Simulation Using Ptolemy II, Ptolemy.org, 2014.
