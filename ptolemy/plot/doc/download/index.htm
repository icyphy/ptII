Download Ptplot

Ptplot is available for download in several format.
Each of the formats contains the same functionality.
  • Tar and Zip files: Instructions
  • 1.24Mb Gzipped Tar File
  • 1.47Mb Compressed Zip File
  • Java Installer tool: Instructions

  • Install via an Applet
  • 1.51Mb Java installer(.class) - To install download and then do 
    	CLASSPATH=.
	export CLASSPATH
	java ptplotinstall

  • 1.52Mb Unix Self Extracting Installer (/bin/sh)
