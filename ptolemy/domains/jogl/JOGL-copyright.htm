Copyright for Java OpenGL (JOGL)

The Jogl domain uses the Java OpenGL (JOGL) interface https://jogl.dev.java.net/. Jogl is BSD licensed, but platform dependent, so we don't include Jogl in the Ptolemy II tree.

Note that configure looks for $PTII/vendors/sun/jogl, so either move the Jogl distribution or create a symbolic link and then rerun configure.

cd $PTII
./configure
