Viptos Demos

These demos require that the AVR tools, TinyOS and nesc be installed.
For details, see the installation notes for successfully running Viptos demos.
  • Sensor To LEDs. Single-node sensing demo with a model of the physical environment.
  • Send and Receive. Two-node radio communication demo with a sending node and a receiving node.
  • Surge. Multihop routing demo with link visualization.
  • Counter To LEDs. May take a moment to open This demo is useful for people who want to create applications and generate nesC code, but not simulate in Viptos.
